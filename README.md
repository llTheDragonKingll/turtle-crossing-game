# 🚗 Turtle Crossing Game (Python)

A simple arcade-style game inspired by Crossy Road, built using Python Turtle graphics.

---

## 🚀 Features

* 👤 Player movement using keyboard controls
* 🚗 Moving car obstacles
* 💥 Collision detection
* 📊 Score tracking and level progression

---

## 🛠️ Tech Stack

* Python
* Turtle Graphics
* Object-Oriented Programming
